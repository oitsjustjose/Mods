Persistent Bits

Simple, Persistent Chunk Loading for Minecraft

Features

Chunk Loader

The Persistent Bits' Chunk Loader is a simple chunk loader with a configurable radius.

The 1.14 re-write of Persistent Bits created a divergence in functionality. The pre-1.14 version uses Forge's Chunk Loading Tickets, and re-initializes the chunk loading process on world join by loading chunk loader tile entity from persistentbits.dat (which ideally you should never delete). The 1.14 version uses the built-in /forceload command logic, and the chunk loaders aren't even tile entities! To ensure that chunks are only loaded as long as a chunk loader is in the main chunk, World Capabilities are used to keep track of how many chunk loaders are in a chunk. As soon as that number hits 0, the force loading is stopped.
