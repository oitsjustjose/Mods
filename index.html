My Collection of Mods

A lot of my mods don't self-document -- this means that the mods themselves don't always have a way of telling you how to use them from within the game. I'm working on this, but it's not possible in every situation unfortunately. That's what this site is for! Think of it as a wiki for the undocumented parts of my mods. Most specifically, V-Tweaks, Persistent Bits and Pulver don't "self-document", and you could argue that Natural Progression's tooltips method isn't robust enough for a user who doesn't know what to look for (and I wouldn't disagree!).

My Code

I keep all my code over at GitHub if you're interested. All of my mod code is free to use so have at it!

About this Site

I'm glad you asked! This site is made using my own scratch-made themekit JuiceUI. This website and the themekit both refuse to use JQuery -- and for no reason in particular! I love and use JQuery often, but it's a nice challenge using just DOM to do everything! Check out the live demo if you want to see what all this themekit is capable of!
